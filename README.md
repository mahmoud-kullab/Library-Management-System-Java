# 📚 Library Management System — Java

A console-based **Library Management System** built with Java, demonstrating core **Object-Oriented Programming (OOP)** principles.

---

## ✨ Features

- 📖 **View all books** — Display the full library catalog with ID, title, and status
- 🔍 **Search** — Find a book by title (case-insensitive)
- 📤 **Borrow** — Borrow an available book
- 📥 **Return** — Return a borrowed book
- ➕ **Add** — Add new books to the library
- 🗑️ **Delete** — Remove a book and auto-reindex IDs

## 🚀 How to Run

1. Make sure you have **Java JDK** installed
2. Download or clone this repository
3. Compile the file:
   ```bash
   javac LibraryManagementSystem.java
   ```
4. Run the program:
   ```bash
   java LibraryManagementSystem
   ```

